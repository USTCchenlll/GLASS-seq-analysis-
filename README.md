# GLASS-seq analysis code

This repository contains custom scripts used for the analysis of GLASS-seq data in the manuscript:

"GLASS-seq: a gel-anchored, ligation-assisted, scalable biosensing platform for low-cost regional spatial transcriptomics"

The code includes scripts for sequencing read parsing, UMI counting, ROI-by-gene matrix generation, normalization, sequencing-depth downsampling, ROI clustering, spatial visualization, and figure generation.
